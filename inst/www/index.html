Bidding Analysis

The purpose of this Bidding Analysis tool is to analyze the performance of rf_codes,mkt_id or other groups of mkt_id over time.Various dimensions-which includes bid placed,predicted rpc of mkt_id,quality,volume, finance,etc of these group are being looked at to analyze present bidding algo and also compare them with other competing algos.

Tools

Each tool consists of a web page having data,plots pertaining to rf_codes,mkt_id or other groups of mkt_id.

  1. Market Id Analysis
  2. Quadratic Modeling
  3. Calculate Errors(RMSE,etc)
  4. A/B Test Results
  5. CID Analysis
  6. Group Mkt Id Analysis
